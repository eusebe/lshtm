
'lshtm' rmarkdown theme for beamer presentation, based on the 'binb' R package and the 'Monash' beamer theme.

Minimal example:

````{md}
---
title: A minimal example
author: Matthias Vogelgesang
date: \today
institute: Centre for Modern Beamer Themes
output: lshtm::lshtm
---

# First Section

## First Frame

Hello, world!
````
